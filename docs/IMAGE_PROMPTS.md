# Image prompts reference

Use these prompts to regenerate or create variations of the Jasmine imagery.

## Auth page background (`auth-bg.png`)

**Prompt:**
> generate a pixelated image of a beautiful lone pink jasmine flower on a green hilly meadow with pretty blue skies

## Hero background (`hero-bg.png`)

**Prompt:**
> now generate it but in an aspect ratio that would be suitable for a hero section. not just one jasmine can be a slightly different design

*(Variation of the auth prompt — multiple jasmine flowers, hero-optimized aspect ratio.)*
